When integrating your infrastructure with Hanzo KMS Cloud, you may need to configure network access controls. This page provides the IP addresses that Hanzo KMS uses to communicate with your services.

## Hanzo KMS IP Addresses

Hanzo KMS Cloud operates from multiple regions. If your infrastructure has strict network policies, you may need to allow traffic from Hanzo KMS by adding the following IP addresses to your ingress rules. These are the IP addresses that Hanzo KMS uses when making outbound requests to your services.

## What These IP Addresses Are Used For

These IP addresses represent the source IPs you'll see when Hanzo KMS Cloud makes connections to your infrastructure. All outbound traffic from Hanzo KMS Cloud originates from these IP addresses, ensuring predictable source IP addresses for your firewall rules.
