---

Prerequisites:

- Set up and add envars to [Hanzo KMS Cloud](https://kms.hanzo.ai)
- [Install the CLI](/cli/overview)

The steps apply to the following non-exhaustive list of frameworks:

- [Express](https://expressjs.com)
- [Fastify](https://www.fastify.io)
- [Koa](https://koajs.com)

## Initialize Hanzo KMS for your app

```bash
# navigate to the root of your of your project 
cd /path/to/project

# then initialize Hanzo KMS
hanzo-kms init
```

## Start your application as usual but with Hanzo KMS

```bash
hanzo-kms run -- <your application start command>

# Example
hanzo-kms run -- npm run dev
```
