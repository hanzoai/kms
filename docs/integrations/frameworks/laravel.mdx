title: "Laravel"
description: "How to use Hanzo KMS to inject environment variables and secrets into a Laravel app."
---

Prerequisites:

- Set up and add envars to [Hanzo KMS Cloud](https://kms.hanzo.ai)
- [Install the CLI](/cli/overview)

## Initialize Hanzo KMS for your [Laravel](https://laravel.com/) app

```bash
# navigate to the root of your of your project 
cd /path/to/project

# then initialize Hanzo KMS
hanzo-kms init
```

## Start your application as usual but with Hanzo KMS

```bash
hanzo-kms run -- <your application start command>

# Example 
hanzo-kms run -- php artisan serve
```
