---
title: "hanzo-kms init"
description: "Switch between Hanzo KMS projects within CLI"
---

```bash
hanzo-kms init
```

## Description

Link a local project to your Hanzo KMS project. Once connected, you can then access the secrets locally from the connected Hanzo KMS project.

<Info>
This command creates a `kms.json` file containing your Project ID.
</Info>
